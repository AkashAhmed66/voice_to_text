Getting Started

The Bangla Audio Transcription API provides powerful speech-to-text conversion for Bangla audio files with AI-powered text refinement.

Base URL

{{ base_url }}

Supported Formats

Audio: WAV, MP3, M4A | Max Size: 50MB

Audio Transcription

POST /api/transcribe

Transcribe Bangla audio file to text with AI enhancement.

cURL Request

curl -X POST {{ base_url }}/api/transcribe \ -F "audio=@your_audio_file.wav" \ -H "Content-Type: multipart/form-data"

Postman Setup

  1. Open Postman and create a new request
  2. Set Method: Select POST from the dropdown
  3. Set URL: Enter {{ base_url }}/api/transcribe
  4. Go to Body tab: Select form-data
  5. Add File Parameter:
    • Key: audio
    • Type: Select File from dropdown
    • Value: Click "Select Files" and choose your audio file
  6. Send Request: Click the blue "Send" button

Success Response

{ "success": true, "data": { "transcription": "আপনার উন্নত বাংলা টেক্সট", "original_transcription": "আপনার মূল বাংলা টেক্সট", "refined": true, "filename": "audio.wav" } }

Error Response

{ "success": false, "error": "No file uploaded" }

Error Codes

HTTP Status Description
200 Success - Audio transcribed successfully
400 Bad Request - Invalid file format, no file uploaded, or file too large
500 Server Error - Processing failed or model unavailable

Usage Tips

  • Audio Quality: Use clear, high-quality recordings for best results
  • File Size: Keep files under 50MB for optimal processing speed
  • Language: API is optimized for Bangla/Bengali language
  • Processing Time: Typical response time is 10-30 seconds depending on audio length
  • AI Enhancement: The API automatically refines transcribed text for better accuracy
